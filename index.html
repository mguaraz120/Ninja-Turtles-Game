Ninja Turtles

You will be given a random number at the start of the game.
each Ninja turtle has a set value that will be hidden. By clicking on them you will add a specifc amount of points to "your total score".
You will win the game by matching your score to the random number. You lose the game if your score
goes above the random number.
Each time when the game starts, all values will be changed.
